# Search Engine (C++20)

Поисковая система для текстовых документов в файловой системе.  
Состоит из двух программ: **Индексатор** строит поисковый индекс по файлам, **Поисковик** выполняет запросы и выдаёт релевантные результаты.

## ⚡ Особенности
- Работает **ИСКЛЮЧИТЕЛЬНО С ЛАТИНСКИМИ БУКВАМИ** (кириллица не поддерживается).
- Для пользователей на Windows имеется build.bat, созданный для автоматизации сборки, но для него потребуется установленный vcpkg и libpqxx соответственно, команда для установки libpqxx через vcpkg "vcpkg install libpqxx:x64-windows" на всякий случай.
- Файлы indexer.exe и searcher.exe буду созданы в папке проекта.
- По времени индексатор работает ровно столько сколько находится файлов в корневой папке, например, если файл один - индексатор быстро завершит работу, а если кроме нескольких файлов есть еще и подпапки, то будет работать дольше. Не рекомендую устанавливать корневую папку C:\\, индексатор отработает, но ждать придется немало, в любом случае все зависит от количества ядер на ПК.
- Многопоточная индексация, обход папок рекурсивно.
- Индексатор сам определяет сколько потоков можно создать.
- Хранение индекса в **PostgreSQL**.
- Консольный интерфейс, запросы до 4 слов, сортировка по сумме частот.
- Кроссплатформенность: используется Cmake.

## 📋 Требования
- **PostgreSQL** (установлен и запущен, в проекте используется 18.1).
- **CMake** 3.20 или новее.
- Компилятор с поддержкой C++20.
- **vcpkg** (для Windows) — для установки `libpqxx`.  
  На Linux можно поставить `libpqxx-dev` из репозитория.

## ⚙️ Настройка `config.ini`
Перед запуском отредактируй `config.ini` в папке проекта:

```ini
# === База данных ===
db_host=localhost
db_port=5432
db_name=имя_вашей_бд
db_user=postgres
db_password=ваш_пароль_от_postgres

# === Индексация ===
- Чтобы указать корневую папку, из которой будет начинаться поиск, нужно в файле `config.ini` найти следующие поля и установить нужные значеня:
- start_directory=нужный_путь (например, . .. или же если нужен конкретный путь то использовать строго такой формат: C:\\users\\desktop\\...)
- file_extensions=перечислите_расширения_файлов_которые_будут_искаться (строго в таком формате, например, txt,log,cpp,md...)
